Sudoku Rules!

A Sudoku puzzle consists of a 9x9 grid composed of 9 inner grids called regions. The puzzle starts out with some of the squares of the 9x9 grid filled in. The ojbective is to fill in the rest of the puzzle according to the following constraints:

  • Each of the squares composing any of the rows must contain any number from 1 through 9. Each row must not have repeat values.
  • Each of the squares composing any of the columns must contain any number from 1 through 9. Each column must not have repeat values.
  • Each of the squares composing any of the regions must contain any number from 1 through 9. Each region must not have repeat values.

That's it! Play a game.
